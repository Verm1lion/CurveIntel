## KONU 1 — Annex G: Elastisite Modülü (E) Hesap Prosedürü

### 1.1 R1 / R2 tanımı (Annex G §G.6.2)

- Annex G, regresyon aralığı için **tavsiye edilen başlangıç değerleri** şöyle verir (birebir alıntı): *"R1 ≈ 10 % of **ReH or Rp0,2**; R2 ≈ 40 % of **ReH or Rp0,2**"*. İfade **"recommended as starting points"** olarak geçer; zorunlu değildir. Alt/üst sınırları kaydırarak yeniden hesaplamaya izin verilir.
- Bazı ikincil kaynaklarda görülen "Rm'nin %10-40'ı" ifadesi **yanlıştır**; oran **ReH veya Rp0,2** üzerinden tanımlıdır.
- Simge tablosu (Table 1): **R1, R2** MPa cinsinden gerilme; alternatif olarak **e1, e2** yüzde cinsinden birim şekil değiştirme kullanılabilir.

### 1.2 Dairesel bağımlılık çözümü (E ↔ Rp0,2)

Standart bu döngüyü iki mekanizma ile çözer:

1. **Ön-test / nominal Rp0,2 kullanımı (birincil yöntem, §G.5.1)**: *"If the stress-strain curve up to ReH or Rp0,2 is not known, a pre-test in advance of the measurement of the modulus of elasticity shall be performed."* Yani üretici spesifikasyonundan veya ön denemeden alınan beklenen Rp0,2 ile R1, R2 seçilir.
- 2. **İteratif düzeltme (§A.3.7)**: *"to get more exact data for Rp0,2, the elastic line should be checked and if necessary recalculated with other limits"* — E bulunduktan sonra 0,2 % offset doğrusu ile Rp0,2 yeniden hesaplanır, gerekirse R1/R2 güncellenip fit tekrarlanır.

TENSTAND WP3 (Lord, Rides, Loveday 2005, NPL DEPC-MPE 016) alternatif olarak **sliding-segment / varyans-minimizasyonu** yaklaşımını önerir: eğrinin en düz bölümü lokal Sm(rel) minimizasyonu ile bulunur ve Rp0,2 bilgisine ihtiyaç kalmaz.

### 1.3 Minimum veri noktası sayısı — **NORMATİF kural**

- §G.3.1.3: *"at least 50 different discrete measured values in the evaluation range"*.
- §G.5.3.2: Örnekleme frekansı alt sınırı için **formül G.1**: $f_{min} = N \cdot \dot{\varepsilon} \cdot E / (R_2 - R_1)$, ile N ≥ 50.
- §G.6.2: *"the number of considered data points should be at least 50"*.

### 1.4 R² eşiği — **standartta açıkça 0,9995**

§G.6.2 birebir: *"the coefficient of determination R², which should be close to 1 (**>0,9995**)"* ve *"R² < 0,9995, the modulus of elasticity should not be determined"*. Yani sadece ticari pratik değil, **standart metninde açık** bir red kriteridir.

### 1.5 Sm(rel) formülü ve eşiği

- **§3.17**: $S_{m,rel} = (S_m / E) \cdot 100\,\%$.
- **§G.6.2 eşiği**: *"It should be less than 1 %"*.
- Sm için kapalı form standartta verilmez; TENSTAND WP3 ve CWA 15261-2 sıradan OLS standart hatasını kullanır:

$$S_m = \sqrt{\frac{1}{n-2}\cdot\frac{\sum_i (R_i - \hat{R}_i)^2}{\sum_i (e_i - \bar{e})^2}}$$

$$R^2 = 1 - \frac{\sum_i(R_i-\hat{R}_i)^2}{\sum_i(R_i-\bar{R})^2}, \qquad m_E = \frac{\sum(e_i-\bar{e})(R_i-\bar{R})}{\sum(e_i-\bar{e})^2}$$

### 1.6 Çok-eğimli / toe bölgesi

Annex G otomatik toe-kaldırma algoritması tanımlamaz. Pratik yaklaşım: (i) §10.2'ye göre ön yük (≤ 5 % Rp0,2) altındaki noktaları at; (ii) çift taraflı ekstensometre ortalaması al (§G.6.1); (iii) R² < 0,9995 olan malzemelerde (örn. dökme demir) **E raporlanmamalıdır** (§G.6.2, açık hüküm).

### 1.7 Tipik E değerleri

| Malzeme | E (GPa) | Kaynak |
|---|---|---|
| Çelik (ferritic/austenitic) | **200–210** | §10.3.3.2 NOTE 2 (standart 210 000 MPa kullanır) |
| Alüminyum alaşımları | **69–73** | Table 3 "<150 000 MPa" sınıfı |
| Titanyum (Ti-6Al-4V) | **~110** | α+β Ti literatürü |

Tipik genişletilmiş belirsizlik U(E): ideal koşulda (çift taraflı Class 0,5 ekstensometre) **±1-2 %**; standart ISO koşullarında **±4-5 %**, kötü koşulda **%14'e** kadar (NPL GPG 98).

### 1.8 Python referans uygulaması

```python
"""
ISO 6892-1:2019 Annex G — Elastisite modülü E hesabı.
Kaynak: §G.3.1.3, §G.5.1, §G.5.3.2, §G.6.2, §A.3.7, §3.15-3.17.
"""
import numpy as np
from dataclasses import dataclass, field

@dataclass
class ModulusResult:
    E_GPa: float
    R2: float
    Sm_rel_pct: float
    n_points: int
    R1_MPa: float
    R2_MPa: float
    Rp02_used_MPa: float
    iterations: int
    status: str
    warnings: list = field(default_factory=list)

def _ols(x, y):
    n = x.size
    xm, ym = x.mean(), y.mean()
    Sxx = np.sum((x - xm) ** 2)
    Sxy = np.sum((x - xm) * (y - ym))
    if Sxx <= 0 or n < 3:
        return np.nan, np.nan, 0.0, np.inf
    m = Sxy / Sxx
    b = ym - m * xm
    yhat = m * x + b
    SSres = np.sum((y - yhat) ** 2)
    SStot = np.sum((y - ym) ** 2)
    R2 = 1.0 - SSres / SStot if SStot > 0 else 0.0
    Sm = np.sqrt(SSres / ((n - 2) * Sxx))
    return m, b, R2, Sm

def estimate_modulus_annexG(stress, strain, Rp02_nominal=None,
                            strain_in_percent=False, max_iter=3,
                            preload_frac=0.02):
    """
    stress: MPa; strain: m/m veya %; Rp02_nominal: ön-test değeri (MPa).
    """
    warnings_list = []
    stress = np.asarray(stress, dtype=float)
    strain = np.asarray(strain, dtype=float)
    if strain_in_percent:
        strain = strain / 100.0

    # Ön tahmin: Rp02 yoksa 0.8 * Rm başlangıç tohumu (§G.5.1 pre-test)
    if Rp02_nominal is None:
        Rp02 = 0.8 * float(np.nanmax(stress))
        warnings_list.append("Rp0,2 verilmedi; 0.8*Rm tohum değeri kullanıldı.")
    else:
        Rp02 = float(Rp02_nominal)

    status = "ok"
    m = np.nan; R2c = 0.0; Sm_rel = np.inf; x = np.array([]); it = 0
    R1 = R2_ = 0.0

    for it in range(1, max_iter + 1):
        R1, R2_ = 0.10 * Rp02, 0.40 * Rp02          # Annex G başlangıç
        # Toe / preload maskesi
        mask = stress >= preload_frac * Rp02
        window = mask & (stress >= R1) & (stress <= R2_)
        x = strain[window]; y = stress[window]

        if x.size < 50:                              # §G.3.1.3 normatif
            status = "few_points"
            warnings_list.append(
                f"Iter {it}: [R1,R2] aralığında {x.size} nokta; ≥50 gerekli.")
            break

        m, b, R2c, Sm = _ols(x, y)
        Sm_rel = 100.0 * Sm / m if m > 0 else np.inf

        # §G.6.2 kabul kapıları
        if R2c < 0.9995:
            status = "low_R2"
            warnings_list.append(
                f"Iter {it}: R²={R2c:.6f} < 0.9995; Annex G'ye göre E "
                "raporlanmamalıdır (malzeme doğrusal olmayabilir).")
        if Sm_rel >= 1.0 and status == "ok":
            status = "high_Sm_rel"
            warnings_list.append(f"Iter {it}: Sm(rel)={Sm_rel:.3f}% ≥ 1%.")

        # 0,2% offset doğrusu ile Rp0,2 iterasyonu
        offset = m * (strain - 0.002)
        diff = stress - offset
        idx = np.where(np.diff(np.sign(diff)) < 0)[0]
        if idx.size == 0:
            break
        Rp02_new = float(stress[idx[0] + 1])
        if abs(Rp02_new - Rp02) / max(Rp02, 1.0) < 0.01:
            Rp02 = Rp02_new
            break
        Rp02 = Rp02_new

    E_GPa = (m / 1000.0) if np.isfinite(m) else np.nan
    return ModulusResult(
        E_GPa=round(E_GPa, 1), R2=R2c, Sm_rel_pct=Sm_rel,
        n_points=int(x.size), R1_MPa=R1, R2_MPa=R2_,
        Rp02_used_MPa=Rp02, iterations=it,
        status=status, warnings=warnings_list)
```

**Kaynaklar:** BS EN ISO 6892-1:2019 Annex G; Gao & Liang 2021 (DOI 10.11973/lhjy-wl202104001); TENSTAND WP3 NPL DEPC-MPE 016; NPL GPG 98 (Lord & Morrell 2007); CWA 15261-2:2005. *Orijinal standart metni, lisanslı ISO kopyasına karşı doğrulanmalıdır (the original standard text should be verified).*

## KONU 2 — Annex A.3.2: ReH/ReL (Süreksiz Akma) Tespiti

### 2.1 Tam algoritma (Annex A.3.2 birebir)

*"ReH should be considered as the stress corresponding to the highest value of the force prior to a reduction of at least **0,5 %** of the force, and followed by a region in which the force should not exceed the previous maximum over a strain range not less than **0,05 %**."*

İki koşul:

$$\text{Koşul 1 (kuvvet düşüşü):}\quad \frac{F_p - F_q}{F_p} \geq 0{,}005$$

$$\text{Koşul 2 (kalıcılık penceresi):}\quad \forall k: e_k - e_q \leq 0{,}05\,\% \Rightarrow F_k \leq F_p$$

Her iki koşulu sağlayan **ilk** lokal maksimum $p$ için: $\boxed{R_{eH} = F_p/S_0}$.

Bu eşik değerleri **ZwickRoell** dokümantasyonu ve **TENSTAND WP2** (NPL DEPC-MPE 015) tarafından birebir doğrulanır.

### 2.2 İki koşullu Python sözde-kodu

```python
def reh_two_condition(F, e_pct, drop=0.005, window_pct=0.05):
    from scipy.signal import find_peaks
    peaks, _ = find_peaks(F, height=0.5 * F.max())
    for p in peaks:
        Fp = F[p]
        drops = (Fp - F[p+1:]) / Fp
        hit = np.where(drops >= drop)[0]
        if not hit.size: continue
        q = p + 1 + hit[0]
        mask = e_pct[q:] <= e_pct[q] + window_pct
        if F[q:][mask].max() <= Fp:
            return p
    return None
```

### 2.3 ReL — "transient masking"

- **§3.10.2.2**: *"ReL — lowest value of stress during plastic yielding, **ignoring any initial transient effects**"*.
- Standart **açık bir sayısal maske değeri vermez**. Sözel ifadedir; TENSTAND bu muğlaklığı sistemik bir sorun olarak belgeler.
- Tedarikçi varsayılanları: **ZwickRoell testXpert** 0,02-0,05 % ötelenme maskesi; **Instron Bluehill** yapılandırılabilir.
- Pratik öneri: Annex A.3.2'nin kendi 0,05 % penceresiyle simetrik olarak **0,05 % maske** uygulanabilir (yapılandırılabilir).

### 2.4 Clause 12 verimlilik kısayolu — **DOĞRU**

§12 birebir: *"ReL may be reported as the **lowest stress within the first 0,25 % strain after ReH**, not taking into account any initial transient effect."* İki normatif kısıt: (i) **Ae belirlenmiyorsa** uygulanabilir; (ii) **test raporunda kaydedilmelidir**.

### 2.5 Karar ağacı (süreksiz vs. Rp0,2)

- Annex A.3.2 iki-koşullu testi **geçerse** → süreksiz akma → ReH, ReL, Ae raporla.
- Geçmezse → sürekli akma → **Rp0,2** (%0,2 offset) raporla.
- §13.1: *"it is inappropriate to determine proof strength during and after discontinuous yielding"* — iki yöntem birbirini dışlar.

**Malzeme rehberi:** Düşük-C yapı çelikleri (S235/S275/S355) ve API 5L (X52-X70) → ReH/ReL; Al alaşımları, östenitik paslanmaz, Cu, Ni alaşımları, Ti → Rp0,2; AlMg alaşımları (AA5182) → **PLC serrated flow** — yukarıdaki mantık kırılır, manuel inceleme gerekir.

### 2.6 Ae (Lüders uzaması) — Clause 16

**Yöntem (a) — yatay çizgi (varsayılan):** son lokal minimum noktadan geçen yatay doğrunun, pekleşme başlangıcındaki **en yüksek eğimli teğet** ile kesişimi.

$$\boxed{\;A_e = \frac{\Delta L_e^{\text{Lüders sonu}} - \Delta L_e^{\text{ReH}}}{L_e}\cdot 100\,\% = e_{LE} - e_{R_{eH}}\;}$$

**Yöntem (b) — regresyon:** akma platosuna OLS doğrusu fit edilir. Seçilen yöntem **rapor edilmelidir**. Ae yuvarlaması 0,1 %'e (§22).

### 2.8 Python referans uygulaması

```python
"""
ISO 6892-1:2019 Annex A.3.2 — ReH/ReL/Ae otomatik tespiti.
"""
from dataclasses import dataclass, field
from typing import Optional, Dict
import numpy as np
from scipy.signal import savgol_filter, find_peaks

DEFAULT_DROP_RATIO = 0.005       # §A.3.2: 0.5 %
DEFAULT_WINDOW_STRAIN = 0.05     # §A.3.2: 0.05 % (strain in %)
DEFAULT_TRANSIENT_MASK = 0.05    # ReL için geçici süreç maskesi
DEFAULT_CLAUSE12_WINDOW = 0.25   # §12 kısayolu

@dataclass
class YieldResult:
    discontinuous: bool
    ReH: Optional[float] = None
    ReL: Optional[float] = None
    Ae: Optional[float] = None
    Rp02: Optional[float] = None
    idx_ReH: Optional[int] = None
    idx_ReL: Optional[int] = None
    idx_end_Luders: Optional[int] = None
    method_Ae: Optional[str] = None
    warnings: list = field(default_factory=list)
    diagnostics: Dict = field(default_factory=dict)

def _find_ReH(F, e, drop, window, res):
    peaks, _ = find_peaks(F, height=0.5 * F.max())
    if peaks.size == 0:
        return None
    if peaks.size > 10:
        res.warnings.append(
            f"{peaks.size} lokal maksimum: olası serrated flow (PLC).")
    N = len(F)
    for p in peaks:
        Fp = F[p]
        drops = (Fp - F[p+1:]) / Fp
        hit = np.where(drops >= drop)[0]
        if not hit.size:
            continue
        q = p + 1 + hit[0]
        e_limit = e[q] + window
        mask = e[q:] <= e_limit
        win = F[q:][mask]
        if win.size and win.max() <= Fp:
            return int(p)
    return None

def _find_ReL_standard(F, e, idx_ReH, mask, res):
    e_start = e[idx_ReH] + mask
    idx_start = int(np.searchsorted(e, e_start))
    if idx_start >= len(F) - 2:
        res.warnings.append("Transient maske veri aralığını aştı.")
        return None
    de = np.gradient(e); dF = np.gradient(F)
    slope = np.divide(dF, de, out=np.zeros_like(dF), where=de > 0)
    smax = np.max(np.abs(slope[idx_ReH:])) + 1e-9
    rising = np.where(slope[idx_start:] > 0.10 * smax)[0]
    idx_end = idx_start + rising[0] if rising.size else len(F) - 1
    if idx_end <= idx_start:
        return None
    return int(idx_start + np.argmin(F[idx_start:idx_end]))

def _find_ReL_clause12(F, e, idx_ReH, window, res):
    e_limit = e[idx_ReH] + window
    idx_end = min(int(np.searchsorted(e, e_limit)), len(F) - 1)
    if idx_end <= idx_ReH + 1:
        return None
    return int(idx_ReH + 1 + np.argmin(F[idx_ReH+1:idx_end]))

def _compute_Ae(e, R, idx_ReH, idx_ReL, res):
    if idx_ReL is None:
        return None, None, None
    idx_stop = min(int(np.searchsorted(e, e[idx_ReL] + 5.0)), len(e) - 1)
    if idx_stop <= idx_ReL + 5:
        return None, None, None
    de = np.gradient(e[idx_ReL:idx_stop])
    dR = np.gradient(R[idx_ReL:idx_stop])
    slope = np.divide(dR, de, out=np.zeros_like(dR), where=de > 0)
    lm = np.where(
        (R[idx_ReL+1:idx_stop-1] < R[idx_ReL:idx_stop-2]) &
        (R[idx_ReL+1:idx_stop-1] < R[idx_ReL+2:idx_stop]))[0]
    if not lm.size:
        return None, None, None
    last_min = idx_ReL + 1 + lm[-1]
    R_base = R[last_min]
    k_max = last_min + int(np.argmax(slope[last_min - idx_ReL:]))
    m = slope[k_max - idx_ReL]
    if m <= 0:
        return None, None, None
    e_LE = e[k_max] + (R_base - R[k_max]) / m
    return int(np.searchsorted(e, e_LE)), "horizontal", float(e_LE - e[idx_ReH])

def _compute_Rp02(e_pct, R, E_MPa, res):
    offset = E_MPa * (e_pct - 0.2) / 100.0
    diff = R - offset
    cross = np.where(np.diff(np.sign(diff)) != 0)[0]
    if not cross.size:
        res.warnings.append("Rp0,2 bulunamadı.")
        return None
    i = cross[0]
    t = diff[i] / (diff[i] - diff[i+1])
    return float(R[i] + t * (R[i+1] - R[i]))

def analyze_yield(strain_pct, force_N, S0_mm2, E_MPa=210_000.0,
                  drop_ratio=DEFAULT_DROP_RATIO,
                  window_strain=DEFAULT_WINDOW_STRAIN,
                  transient_mask=DEFAULT_TRANSIENT_MASK,
                  sg_window=21, sg_order=3,
                  use_clause12_shortcut=False):
    e = np.asarray(strain_pct, float)
    F = np.asarray(force_N, float)
    res = YieldResult(discontinuous=False)
    if sg_window % 2 == 0: sg_window += 1
    F_s = savgol_filter(F, sg_window, sg_order) if len(F) >= sg_window else F
    R = F / S0_mm2; R_s = F_s / S0_mm2

    idx = _find_ReH(F_s, e, drop_ratio, window_strain, res)
    if idx is None:
        res.Rp02 = _compute_Rp02(e, R_s, E_MPa, res)
        res.warnings.append("Süreksiz akma yok; Rp0,2 raporla.")
        return res

    res.discontinuous = True
    res.idx_ReH = idx
    res.ReH = float(R[idx])

    idx_ReL = (_find_ReL_clause12(F_s, e, idx, DEFAULT_CLAUSE12_WINDOW, res)
               if use_clause12_shortcut
               else _find_ReL_standard(F_s, e, idx, transient_mask, res))
    if idx_ReL is not None:
        res.idx_ReL = idx_ReL
        res.ReL = float(R[idx_ReL])
    if use_clause12_shortcut:
        res.warnings.append("Clause 12 kısayolu — test raporunda belirtin.")

    idx_end, method, Ae = _compute_Ae(e, R_s, idx, idx_ReL, res)
    res.idx_end_Luders, res.method_Ae, res.Ae = idx_end, method, Ae

    if res.ReH and res.ReL:
        res.diagnostics["ReH_over_ReL"] = res.ReH / res.ReL
    return res
```

## KONU 3 — Annex A.3.6.1: Fracture (At) Otomatik Tespiti

### 3.1 Tam algoritma (birebir)

*"A.3.6.1 Determine At with reference to the definition of fracture in Figure A.2. The fracture is considered to be effective when the force between two consecutive points decreases: (a) by more than five times the difference between the value of the previous two points, followed by a decrease to lower than 2 % of the maximum tensile force, **and** (b) lower than 2 % of the maximum tensile force (soft materials)."*

Figure A.2 anahtarı: $|\Delta F_{n+1,n}| > 5\,|\Delta F_{n,n-1}|$ **"and/or"** $F_{n+1} < 0{,}02\,F_m$.

$$\boxed{\text{Kriter 1:}\ |\Delta F_{n+1,n}| > 5\cdot|\Delta F_{n,n-1}|}\quad \boxed{\text{Kriter 2:}\ F_{n+1} < 0{,}02\,F_m}$$

$$A_t = \frac{\Delta L_f}{L_e}\cdot 100\,\%$$

### 3.2 AND mi OR mu? — Kritik sorunun cevabı

Standart içinde **fiili bir tutarsızlık** vardır: metin "and" der ama (a) zaten 2 % koşulunu içeren bileşik bir ifadedir; Figure A.2 anahtarı "**and/or**" kullanır. Doğru mühendislik yorumu:

- **Kırılgan/ani malzemeler** için: Kriter (a) — 5× ani düşüş **VE** arkasından 2 %'nin altına iniş (bileşik AND).
- **Yumuşak/sünek malzemeler** için: Kriter (b) — salt 2 %·Fm eşiği (hiçbir zaman 5× ani düşüş olmaz).
- İki kural **OR** mantığıyla birlikte çalışır: hangisi önce tetiklenirse kırılma anı odur. Ticari yazılımlar (ZwickRoell testXpert, Instron Bluehill, MTS TestSuite) bu yorumu uygular.

Uygulama önerisi: varsayılan `logic="ISO"` (OR-of-[compound-a, standalone-b]); strict AND opsiyonel.

### 3.3 Sünek vs. kırılgan

Standart açık bir sınıflandırma yapmaz; ancak **"(soft materials)"** açıklaması Kriter (b)'nin sünek malzemelere yönelik olduğunu gösterir. Kırılgan malzemeler için (dökme demir, martenzitik çelik) Kriter (a) önce tetiklenir; sünek için (düşük-C çelik, Al, östenitik paslanmaz) sadece (b).

### 3.4 Adlandırma: At vs. A vs. Af

| Sembol | Tanım | Kaynak |
|---|---|---|
| **At** | Kırılmada toplam yüzde uzama (eğri üstünden, elastik dahil) | §3.6.6 — Annex A.3.6.1'in ürettiği büyüklük |
| **A** | Kırılma sonrası yüzde kalıcı uzama (fiziksel ölçüm, $(L_u-L_0)/L_0$) | §3.4.2, §20 |
| **Ag** | Fmax'taki plastik uzama (elastik çıkarılmış) | §3.6.5, §17 Formula (3) |
| **Agt** | Fmax'taki toplam uzama | §3.6.4, §18 |
| **Af** | **ISO 6892-1'DE YOK** | — |

Kuvvet-düşüş algoritmasının ürettiği değer resmi olarak **At**'tır; "Af" standart-dışı satıcı/informel terimdir, uyumlu raporlarda kullanılmamalıdır.

**A ↔ At ilişkisi:** $A \approx A_t - (R_m/m_E)\cdot 100\,\%$ — elastik geri yayın çıkarılır (§20.2).

### 3.5 Diğer özellikler

$$R_m = F_m/S_0,\qquad A_g = (\Delta L_m/L_e - R_m/m_E)\cdot 100,\qquad A_{gt} = (\Delta L_m/L_e)\cdot 100,\qquad Z = (S_0 - S_u)/S_0 \cdot 100$$

### 3.6 Kenar durumlar

- **Gürültü:** Annex A.3.6.1 uyarır: *"An increased sampling rate and/or filtering of the force signal may influence the point of fracture"*. Savitzky-Golay öncesi yumuşatma önerilir.
- **Fmax öncesi sahte tetikleme:** `require_past_Fm=True` zorunluluğu; kırılma fiziksel olarak Fmax sonrasında olur.
- **Kademeli kırılma (çoklu düşüş):** `persistence` parametresi — sonraki birkaç örnek de 2 %·Fm altında kalmalı.
- **Veri kırılmadan önce sonlanırsa:** `fracture_detected=False` tanısı, uyarı ile birlikte son noktanın At'ı opsiyonel raporlanabilir.
- **Ekstensometre kırılma öncesi çıkarıldıysa:** §A.3.6.3 crosshead yer değiştirmesiyle tamamlamaya izin verir.

```python
"""
ISO 6892-1:2019 Annex A.3.6.1 — At, Ag, Agt, Rm otomatik hesabı.
"""
from __future__ import annotations
from dataclasses import dataclass, field
from typing import Optional, Literal
import numpy as np
from scipy.signal import savgol_filter

@dataclass
class TensileResult:
    At: Optional[float] = None
    Ag: Optional[float] = None
    Agt: Optional[float] = None
    Rm: Optional[float] = None
    Fm: Optional[float] = None
    idx_Fm: Optional[int] = None
    idx_fracture: Optional[int] = None
    dLf: Optional[float] = None
    material_class: Literal["brittle","ductile","unknown"] = "unknown"
    criterion_triggered: Literal["A(abrupt+2%)","B(soft-2%)","none"] = "none"
    fracture_detected: bool = False
    validity_notes: list = field(default_factory=list)

def _smooth(x, window=11, poly=2):
    if len(x) < window or window < 5: return x.copy()
    if window % 2 == 0: window += 1
    return savgol_filter(x, window, poly)

def detect_fracture_iso6892(force, extension, Le, S0=None, mE=None,
                            logic: Literal["ISO","OR","AND"]="ISO",
                            smooth_window=11, require_past_Fm=True,
                            persistence=1):
    """
    force [N], extension [mm, ΔL], Le [mm], S0 [mm^2], mE [MPa/%].
    """
    F = np.asarray(force, float)
    E = np.asarray(extension, float)
    res = TensileResult()
    if F.size != E.size or F.size < 5:
        res.validity_notes.append("Girdi dizileri çok kısa veya uyumsuz.")
        return res
    Fs = _smooth(F, smooth_window) if smooth_window > 2 else F.copy()
    idx_Fm = int(np.argmax(Fs)); Fm = float(Fs[idx_Fm])
    res.idx_Fm = idx_Fm; res.Fm = Fm
    if S0: res.Rm = Fm / S0
    dLm = float(E[idx_Fm])
    res.Agt = 100.0 * dLm / Le
    if mE is not None and res.Rm is not None:
        res.Ag = res.Agt - (res.Rm / mE) * 100.0

    start = idx_Fm + 1 if require_past_Fm else 2
    end = len(Fs) - 1
    if start >= end:
        res.validity_notes.append("Fmax sonrası örnek yok.")
        return res
    thr_soft = 0.02 * Fm
    dF = np.diff(Fs)
    idx_fr = None; trig = "none"

    for n in range(start, end):
        prev = abs(dF[n-1]) if n-1 >= 0 else 0.0
        curr = abs(dF[n])
        # Kriter 1: 5× düşüş (sadece negatif yönde)
        crit1 = (curr > 5.0 * prev) and (dF[n] < 0)
        # Kriter 2: sonraki 'persistence' örnek de eşik altında
        pers_end = min(n + 1 + persistence, len(Fs))
        crit2 = bool(np.all(Fs[n+1:pers_end] < thr_soft))

        if logic == "ISO":
            if crit1 and crit2:
                idx_fr, trig = n+1, "A(abrupt+2%)"; break
            if crit2:
                idx_fr, trig = n+1, "B(soft-2%)"; break
        elif logic == "OR":
            if crit1 or crit2:
                idx_fr = n+1
                trig = "A(abrupt+2%)" if crit1 else "B(soft-2%)"; break
        elif logic == "AND":
            if crit1 and crit2:
                idx_fr, trig = n+1, "A(abrupt+2%)"; break

    if idx_fr is None:
        res.validity_notes.append(
            "Kırılma tespit edilemedi; test erken bitmiş olabilir.")
        res.At = 100.0 * float(E[-1]) / Le
        res.dLf = float(E[-1]); res.idx_fracture = len(Fs)-1
        res.fracture_detected = False
        res.material_class = "unknown"
    else:
        res.dLf = float(E[idx_fr])
        res.At = 100.0 * res.dLf / Le             # §19 Formula (5)
        res.idx_fracture = idx_fr
        res.criterion_triggered = trig
        res.fracture_detected = True
        post = idx_fr - idx_Fm; total = len(Fs) - idx_Fm
        if trig == "A(abrupt+2%)" and total > 0 and post/total < 0.20:
            res.material_class = "brittle"
        else:
            res.material_class = "ductile"

    if Le <= 0: res.validity_notes.append("Le pozitif olmalı.")
    if S0 is None: res.validity_notes.append("S0 verilmedi; Rm hesaplanmadı.")
    if mE is None: res.validity_notes.append("mE verilmedi; Ag hesaplanmadı.")
    return res
```

### 4.4 Rp0,2 birleşik belirsizliği (GUM)

Model: $R_{p0,2} = F_{p0,2}/S_0$. Göreli form:

$$\boxed{\;\left(\frac{u_c(R_{p0,2})}{R_{p0,2}}\right)^2 = \left(\frac{u(F_{p0,2})}{F_{p0,2}}\right)^2 + \left(\frac{u(S_0)}{S_0}\right)^2 + \sum_i \left(\frac{c_i\,u(x_i)}{R_{p0,2}}\right)^2 + \frac{s^2_{R_{p0,2}}}{n}\;}$$

Dairesel kesit: $u(S_0)/S_0 = 2\,u(d_0)/d_0$. Dikdörtgen kesit: $(u(S_0)/S_0)^2 = (u(a_0)/a_0)^2 + (u(b_0)/b_0)^2$.

$F_{p0,2}$ ayrıca E ve Le üzerinden offset-doğrusu kesişiminden gelen ek belirsizlik taşır (UNCERT COP 07 Denklem 42-58).

### 4.5 Rm, E, A birleşik belirsizlikleri

$$\boxed{\;\left(\frac{u_c(R_m)}{R_m}\right)^2 = \left(\frac{u(F_m)}{F_m}\right)^2 + \left(\frac{u(S_0)}{S_0}\right)^2 + \frac{s^2_{R_m}}{n}\;}$$

$$\boxed{\;\left(\frac{u_c(E)}{E}\right)^2 = \left(\frac{u(m_E)}{m_E}\right)^2 + \left(\frac{u(L_0)}{L_0}\right)^2 + \left(\frac{u(S_0)}{S_0}\right)^2\;}\qquad u(m_E) = S_m = \sqrt{\frac{SS_{res}}{(n-2)\,S_{xx}}}$$

$$u_c^2(A) = \left(\frac{100}{L_0}\right)^2 u^2(L_u) + \left(\frac{100\,L_u}{L_0^2}\right)^2 u^2(L_0) + u^2(C_{A,m}) + s_A^2/n$$

### 4.6 Genişletilmiş belirsizlik

$$U = k\cdot u_c,\qquad k = 2\ (%95,45)$$

Küçük n (< 10) için Welch-Satterthwaite ile efektif serbestlik derecesi hesaplanır, k = $t_{0,975,\nu_{eff}}$ (örn. n = 5 → ν = 4 → k ≈ 2,776). **GUM uyumludur.**

```python
"""
ISO 6892-1:2019 Annex K + GUM JCGM 100:2008 ölçüm belirsizliği bütçesi.
"""
from __future__ import annotations
import math
from dataclasses import dataclass, field
from typing import Dict, List, Optional
import numpy as np
from scipy import stats

@dataclass
class SensorUncertainty:
    u_rel_F: float = 0.005
    u_rel_Le: float = 0.005
    u_rel_L0: float = 0.002
    u_rel_a0: float = 0.002
    u_rel_b0: float = 0.002
    u_rel_align: float = 0.002
    u_rel_T: float = 0.001
    u_rel_rate: float = 0.003
    u_rel_soft_E: float = 0.02

@dataclass
class TestResult:
    F_max: float; F_p02: float; S0: float; L0: float
    Lu: Optional[float] = None; Su: Optional[float] = None
    E: Optional[float] = None; u_E_reg: Optional[float] = None

@dataclass
class UncertaintyComponent:
    name: str; value: float; typeAB: str; dof: float = math.inf

@dataclass
class Budget:
    measurand: str; value: float; unit: str
    components: List[UncertaintyComponent] = field(default_factory=list)
    u_c: float = 0.0; U_exp: float = 0.0; k: float = 2.0
    dof_eff: float = math.inf

    def percent_contributions(self):
        tot = sum(c.value**2 for c in self.components)
        return {c.name: 100.0*c.value**2/tot for c in self.components} if tot else {}

def _welch_satterthwaite(comps, u_c):
    denom = sum(c.value**4/c.dof for c in comps
                if c.dof > 0 and math.isfinite(c.dof))
    return math.inf if denom == 0 else u_c**4/denom

def _coverage_k(dof, p=0.9545):
    if not math.isfinite(dof) or dof >= 30: return 2.0
    return float(stats.t.ppf(0.5 + p/2.0, dof))

def combine(measurand, value, unit, comps, p=0.9545):
    if value == 0 or not math.isfinite(value):
        raise ValueError(f"{measurand}: geçersiz değer.")
    u_c = math.sqrt(sum(c.value**2 for c in comps))
    dof = _welch_satterthwaite(comps, u_c)
    k = _coverage_k(dof, p)
    return Budget(measurand, value, unit, comps, u_c, k*u_c, k, dof)

def budget_Rm(res, s, replicates=None):
    Rm = res.F_max / res.S0
    comps = [
        UncertaintyComponent('u(F_max)', s.u_rel_F*Rm, 'B'),
        UncertaintyComponent('u(S0)', s.u_rel_a0*Rm*math.sqrt(2), 'B'),
        UncertaintyComponent('u(align)', s.u_rel_align*Rm, 'B'),
        UncertaintyComponent('u(T)', s.u_rel_T*Rm, 'B'),
        UncertaintyComponent('u(rate)', s.u_rel_rate*Rm, 'B')]
    if replicates is not None and len(replicates) >= 2:
        n = len(replicates)
        sA = float(np.std(replicates, ddof=1)/math.sqrt(n))
        comps.append(UncertaintyComponent('u_A(repeat)', sA, 'A', dof=n-1))
    return combine('Rm', Rm, 'MPa', comps)

def budget_Rp02(res, s, replicates=None):
    Rp = res.F_p02 / res.S0
    comps = [
        UncertaintyComponent('u(F_p02)', s.u_rel_F*Rp, 'B'),
        UncertaintyComponent('u(S0)', s.u_rel_a0*Rp*math.sqrt(2), 'B'),
        UncertaintyComponent('u(Le)', s.u_rel_Le*Rp, 'B'),
        UncertaintyComponent('u(L0)', s.u_rel_L0*Rp, 'B'),
        UncertaintyComponent('u(align)', s.u_rel_align*Rp, 'B'),
        UncertaintyComponent('u(rate)', s.u_rel_rate*Rp, 'B'),
        UncertaintyComponent('u(T)', s.u_rel_T*Rp, 'B')]
    if replicates is not None and len(replicates) >= 2:
        n = len(replicates)
        sA = float(np.std(replicates, ddof=1)/math.sqrt(n))
        comps.append(UncertaintyComponent('u_A(repeat)', sA, 'A', dof=n-1))
    return combine('Rp0.2', Rp, 'MPa', comps)

def budget_E(res, s, replicates=None):
    if res.E is None:
        raise ValueError("E Annex G regresyonundan verilmelidir.")
    u_reg = res.u_E_reg if res.u_E_reg is not None else s.u_rel_soft_E*res.E
    comps = [
        UncertaintyComponent('u(m_E)[reg]', u_reg, 'A', dof=48),
        UncertaintyComponent('u(L0)', s.u_rel_L0*res.E, 'B'),
        UncertaintyComponent('u(S0)', s.u_rel_a0*res.E*math.sqrt(2), 'B'),
        UncertaintyComponent('u(align)', s.u_rel_align*res.E, 'B'),
        UncertaintyComponent('u(Le)', s.u_rel_Le*res.E, 'B'),
        UncertaintyComponent('u(soft)', s.u_rel_soft_E*res.E, 'B')]
    if replicates is not None and len(replicates) >= 2:
        n = len(replicates)
        sA = float(np.std(replicates, ddof=1)/math.sqrt(n))
        comps.append(UncertaintyComponent('u_A(repeat)', sA, 'A', dof=n-1))
    return combine('E', res.E, 'GPa', comps)

def budget_A(res, s, u_Lu_abs=0.05, u_L0_abs=0.05, replicates=None):
    if res.Lu is None: raise ValueError("Lu gerekli.")
    A = (res.Lu - res.L0) * 100.0 / res.L0
    cLu = 100.0/res.L0; cL0 = -100.0*res.Lu/(res.L0**2)
    comps = [
        UncertaintyComponent('u(Lu)', abs(cLu)*u_Lu_abs, 'B'),
        UncertaintyComponent('u(L0)', abs(cL0)*u_L0_abs, 'B'),
        UncertaintyComponent('u(C_Am)', 0.2, 'B')]
    if replicates is not None and len(replicates) >= 2:
        n = len(replicates)
        sA = float(np.std(replicates, ddof=1)/math.sqrt(n))
        comps.append(UncertaintyComponent('u_A(repeat)', sA, 'A', dof=n-1))
    return combine('A', A, '%', comps)

def report(b):
    lines = [f"=== {b.measurand} = {b.value:.4g} {b.unit} ===",
             f"  u_c = {b.u_c:.4g} {b.unit} ({100*b.u_c/b.value:.2f} %)",
             f"  dof_eff = {b.dof_eff:.1f}  k = {b.k:.3f}",
             f"  U(k={b.k:.2f}) = {b.U_exp:.4g} {b.unit} "
             f"({100*b.U_exp/b.value:.2f} %)", "  Bileşenler:"]
    contrib = b.percent_contributions()
    for c in sorted(b.components, key=lambda x: -x.value):
        lines.append(f"    {c.name:16s} ({c.typeAB})  "
                     f"u = {c.value:9.4g}  katkı = {contrib[c.name]:5.1f} %")
    return "\n".join(lines)
```

## KONU 5 — Dixon Q Test: Kritik Değerler ve Uygulama

### 5.1 Dixon varyantları ($r_{10}, r_{11}, r_{21}, r_{22}$)

Sıralı veri $x_1 \leq x_2 \leq \dots \leq x_n$ için:

| İstatistik | n aralığı | Üst-şüpheli ($x_n$) | Alt-şüpheli ($x_1$) |
|---|---|---|---|
| **$r_{10}$ (Q)** | 3 ≤ n ≤ 7 | $(x_n - x_{n-1})/(x_n - x_1)$ | $(x_2 - x_1)/(x_n - x_1)$ |
| **$r_{11}$** | 8 ≤ n ≤ 10 | $(x_n - x_{n-1})/(x_n - x_2)$ | $(x_2 - x_1)/(x_{n-1} - x_1)$ |
| **$r_{21}$** | 11 ≤ n ≤ 13 | $(x_n - x_{n-2})/(x_n - x_2)$ | $(x_3 - x_1)/(x_{n-1} - x_1)$ |
| **$r_{22}$** | 14 ≤ n ≤ 30 | $(x_n - x_{n-2})/(x_n - x_3)$ | $(x_3 - x_1)/(x_{n-2} - x_1)$ |

Varyant seçimi **maskelenme**yi (ikinci bir aykırı değerin paydayı büyütüp testi nötrleştirmesini) azaltır. Hipotez reddi: $Q_{calc} > Q_{krit}(\alpha, n)$.

### 5.2 Kritik değerler tablosu — Rorabacher 1991 (Anal. Chem. 63:139-146)

$r_{10}$ istatistiği için iki taraflı kritik değerler:

| n | $Q_{0,10}$ | $Q_{0,05}$ | $Q_{0,01}$ |
|---|---|---|---|

Stat. Softw. 16:3) ve Verma & Quiroz-Ruiz 2006'dan alınmalıdır; Rorabacher tek başına $r_{10}$ tablosunu verir.

### 5.3 Dixon vs. Grubbs — küçük n avantajı

- **Grubbs** $G = (x_{süpheli} - \bar{x})/s$: aynı küçük örnekten hem ortalamayı hem standart sapmayı kestirir; n = 3'te tek aykırı değer s'yi öyle şişirir ki G kritik değere ulaşamaz (**"aykırı değer varyansın içinde saklanır"**).
- **Dixon** sadece sıra istatistiklerini (aralık oranı) kullanır; s'yi hesaplamaz; **kendi kendini maskelemez**. n ≤ 7'de tercih edilir.
- n ≥ 8-10'da Grubbs daha güçlüdür; n > 30'da Generalized ESD / Tietjen-Moore önerilir.
- Her iki test de yaklaşık **Gauss ana kütle** varsayar. Çekme-mukavemeti verileri genellikle Weibull (zayıf halka) dağılımı gösterir; şüpheli dışlanmış şekilde Shapiro-Wilk normallik testi yapılmalıdır.

### 5.4 İteratif uygulama — **ÖNERİLMEZ**

Dean & Dixon 1951 açıkça: *"this test should be used sparingly and **never more than once** in a data set"*. Nedenler:

- **Ardışık p-değeri bozulması**: k iterasyon sonrası aile-yönünde Tip-I hata $\approx 1 - (1-\alpha)^k$.
- **Maskelenme/süpürme**: İki benzer aykırı birbirini maskeler; bir tanesi çıkarıldıktan sonra meşru değerler yanlışlıkla aykırı görünür.
- Bunun yerine n ≥ 8 için $r_{11}/r_{21}/r_{22}$ varyantlarını kullanın (maskelenmeye karşı tasarlanmışlardır).

### 5.5 ISO çerçevesi — Dixon'ın yeri

- **ISO 5725-2:2019** (ölçüm metodu kesinliği): Mandel h/k grafikleri → **Cochran** (varyans aykırıları) → **Grubbs** (ortalama aykırıları, tek ve çift). α=5 % "straggler", α=1 % "outlier" olarak etiketlenir; **otomatik silinmez**. Dixon ismen anılmaz ama 2019 revizyonu dokümante edilmek koşuluyla alternatiflere izin verir.
- **ISO/IEC 17025:2017**: Belirli bir test dayatmaz; sadece **dokümante prosedür** ister (Clause 7.5, 7.7, 7.8, 7.10). Ham veri (ret edilenler dahil) saklanmalıdır.
- **EDQM OMCL PA/PH/OMCL(13)113R7**: *"the rejection of observations solely because a test for outliers is significant is discouraged"* — aykırı değer testi kök-neden analizinin yerine geçmez.

### 5.6 Çekme testi bağlamı (n = 3 veya 5)

**n = 3 (üçlü):**
- Kritik değerler: 0,941 (α=0,10), 0,970 (α=0,05), 0,994 (α=0,01).
- Boşluk tam aralığın %97'sinden büyük olmalı → yalnızca 3-4σ aykırılar yakalanabilir.
- Grubbs n=3'te matematiksel tavan 1,155'tir; pratikte işe yaramaz.
- **Öneri:** Dixon matematiksel olarak uygulanabilir fakat güç düşüktür; mümkünse n=5'e çıkın.

**n = 5 (tercih edilen çekme parti boyutu):**
- Kritik değerler: 0,642 / 0,710 / 0,821.
- ≈2,5σ aykırı değeri yakalayabilir; yeterli güç.
- Dixon açıkça Grubbs'tan üstündür.

### 5.7 Python referans uygulaması

```python
"""
Dixon Q testi (r10/r11/r21/r22 otomatik seçimli) + Grubbs fallback.
Çekme testi replika QC için; tek-geçiş, iteratif uygulanmaz.
Kritik değerler: Rorabacher (Anal. Chem. 1991, 63:139-146).
"""
from __future__ import annotations
from dataclasses import dataclass
from typing import Literal
import warnings, math
import numpy as np
from scipy import stats

_DIXON_Q_TABLE = {
    3:(0.941,0.970,0.994), 4:(0.765,0.829,0.926),
    5:(0.642,0.710,0.821), 6:(0.560,0.625,0.740),
    7:(0.507,0.568,0.680), 8:(0.468,0.526,0.634),
    9:(0.437,0.493,0.598), 10:(0.412,0.466,0.568),
    11:(0.392,0.444,0.542), 12:(0.376,0.426,0.522),
    13:(0.361,0.410,0.503), 14:(0.349,0.396,0.488),
    15:(0.338,0.384,0.475), 16:(0.329,0.374,0.463),
    17:(0.320,0.365,0.452), 18:(0.313,0.356,0.442),
    19:(0.306,0.349,0.433), 20:(0.300,0.342,0.425),
    21:(0.295,0.337,0.418), 22:(0.290,0.331,0.411),
    23:(0.285,0.326,0.404), 24:(0.281,0.321,0.399),
    25:(0.277,0.317,0.393), 26:(0.273,0.312,0.388),
    27:(0.269,0.308,0.384), 28:(0.266,0.305,0.380),
    29:(0.263,0.301,0.376), 30:(0.260,0.290,0.372),
}
_ALPHA_COL = {0.10: 0, 0.05: 1, 0.01: 2}

@dataclass
class DixonResult:
    variant: str; n: int; alpha: float
    Q_calc: float; Q_critical: float
    is_outlier: bool; suspect_value: float
    position: Literal["min","max","none"]
    note: str = ""

def dixon_q_test(data, alpha=0.05,
                 side: Literal["two-sided","min","max"]="two-sided",
                 strict_variant=True) -> DixonResult:
    if alpha not in _ALPHA_COL:
        raise ValueError(f"alpha ∈ {list(_ALPHA_COL.keys())}")
    x = np.asarray(data, float).ravel()
    x = x[~np.isnan(x)]
    n = x.size
    if n < 3:
        raise ValueError("Dixon Q testi n≥3 gerektirir.")
    if n > 30:
        warnings.warn("n>30: Rorabacher tablosu dışı; Grubbs'a geçiliyor.")
        return grubbs_test(x, alpha=alpha, side=side)
    s = np.sort(x)
    if s[-1] - s[0] == 0:
        return DixonResult("r10", n, alpha, 0.0,
                           _DIXON_Q_TABLE[n][_ALPHA_COL[alpha]],
                           False, float(s[0]), "none",
                           note="Sabit veri; aykırı yok.")
    if strict_variant:
        if   3<=n<=7:  variant,(j_lo,k_lo,j_hi,k_hi)="r10",(1,-1,-2, 0)
        elif 8<=n<=10: variant,(j_lo,k_lo,j_hi,k_hi)="r11",(1,-2,-2, 1)
        elif 11<=n<=13:variant,(j_lo,k_lo,j_hi,k_hi)="r21",(2,-2,-3, 1)
        else:          variant,(j_lo,k_lo,j_hi,k_hi)="r22",(2,-3,-3, 2)
    else:
        variant,(j_lo,k_lo,j_hi,k_hi)="r10",(1,-1,-2, 0)
    Q_low  = (s[j_lo] - s[0])  / (s[k_lo] - s[0])
    Q_high = (s[-1]  - s[j_hi]) / (s[-1]  - s[k_hi])
    Q_crit = _DIXON_Q_TABLE[n][_ALPHA_COL[alpha]]
    if side == "min":
        Q_calc, pos, suspect = Q_low, "min", float(s[0])
    elif side == "max":
        Q_calc, pos, suspect = Q_high, "max", float(s[-1])
    else:
        if Q_high >= Q_low:
            Q_calc, pos, suspect = Q_high, "max", float(s[-1])
        else:
            Q_calc, pos, suspect = Q_low, "min", float(s[0])
    note = "Şüpheli komşusuyla eşit; aykırı olamaz." if Q_calc == 0.0 else ""
    return DixonResult(variant, n, alpha, float(Q_calc), float(Q_crit),
                       bool(Q_calc > Q_crit), suspect, pos, note)

def grubbs_test(data, alpha=0.05,
                side: Literal["two-sided","min","max"]="two-sided"):
    x = np.asarray(data, float).ravel()
    x = x[~np.isnan(x)]
    n = x.size
    if n < 3: raise ValueError("Grubbs n≥3.")
    mean = x.mean(); s = x.std(ddof=1)
    if s == 0:
        return DixonResult("grubbs", n, alpha, 0.0, float("inf"),
                           False, float(mean), "none", note="Sıfır varyans.")
    t2 = stats.t.ppf(1 - alpha/(2*n), n-2) ** 2
    G_crit = ((n-1)/math.sqrt(n)) * math.sqrt(t2/(n-2+t2))
    dev_max = x.max() - mean; dev_min = mean - x.min()
    if side == "min" or (side == "two-sided" and dev_min > dev_max):
        G_calc, pos, suspect = dev_min/s, "min", float(x.min())
    else:
        G_calc, pos, suspect = dev_max/s, "max", float(x.max())
    return DixonResult("grubbs", n, alpha, float(G_calc), float(G_crit),
                       bool(G_calc > G_crit), suspect, pos)

def qc_flag_outlier(values, alpha=0.05,
                    method: Literal["auto","dixon","grubbs"]="auto"):
    x = np.asarray(values, float).ravel()
    x = x[~np.isnan(x)]
    n = x.size
    if n < 3:
        raise ValueError("Aykırı testi için en az 3 replika gerekli.")
    if method == "grubbs" or (method == "auto" and n > 30):
        return grubbs_test(x, alpha=alpha)
    return dixon_q_test(x, alpha=alpha, strict_variant=(method != "dixon"))
```
