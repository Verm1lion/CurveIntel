# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in CurveIntel, please report it responsibly:

1. **Do NOT** open a public GitHub issue
2. Email the maintainer directly with details of the vulnerability
3. Include steps to reproduce the issue
4. Allow reasonable time for a fix before public disclosure

## Scope

CurveIntel is a data analysis tool. Security concerns primarily relate to:

- Input validation (malicious CSV/Excel files)
- Data integrity (calculation accuracy)
- Authentication and authorization (web dashboard)
- Audit trail integrity
